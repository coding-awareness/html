Class

The class attribute used to refer css style for HTML element. Multiple elements can have same class name. Also, Used in JavaScript to identify elements for modification purpose.

It's case sensitive, But recommended to write in small case. With dot(.) we can write css style.

Id

The id attribute is used to speficy unique identify for element. There shouldn't be same id for another element in a HTML page.

It's case sensitive, with hash(#) we can css style.
