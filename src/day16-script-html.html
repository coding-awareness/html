What is Script?

Using script tag we can include programming into HTML. So, It will make web page interactive.

The scripting languages are JavaScript and VBScript. But, VBScript is only supported by Internet Explorer browser. So, We are going to use JavaScript.

Two way to include script in HTML:

  • Internal - Inside script tag we can write JavaScript.
  • External - Using script tag's src attribute we can include external .js file
