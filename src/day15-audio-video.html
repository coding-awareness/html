What is Audio tag?

Audio element used to play music in web page.
It's attribute controls provide play, pause and stop controls.

The source element used to specify the file inside audio tag.
Multiple source elements can be added with different types of format. Browser will get first source's file.

What is Video tag?

Video element used to play music in web page.
It's attribute controls provide play, pause and stop controls.
The autoplay allows to play media automatically. But some browsers won't allow autoplay without muted attribute.
width and height will help to place video properly.

The source element used to specify the file inside audio tag.
Multiple source elements can be added with different types of format. Browser will get first source's file.
