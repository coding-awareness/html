What is Style?

Set of rules that describes how document will be display in a browser.

Styles can be added in 3 ways:

  1. External - A separate file
  2. Embedded - Using style element within the head tag
  3. Inline - Using style attribute on any visible elements within body tag

Style rules are property value pairs with selectors:
Selectors is to write style rules for specific elements 

 
        p (Selector) {
            color: red; (property: value)
            font-size: 14px;
            background-color: green;
        }
